SkillTreat Native

SkillTreat is a native, full-featured, cross-platform mobile app made with the React Native framework. It features user identity management, account registration, cart management, and full integration with the Stripe API. It serves as the front end for the SkillTreat API.

SkillTreat API

SkillTreat is an eCommerce API built with PostgreSQL, Sequelize, NodeJs, and Express. It's deployed on AWS and uses SSL encryption. Since SkillTreat was built for in-person classes that take place at a specific point in time, it goes beyond typical eComm functionality in that each purchasable class has start and end times, which gate a lot of functionality (e.g., a user can't see or sign up for a class that already started).

markWith

markWith is a collaborative Markdown editor (and API) that allows multiple users to work on the same Markdown file together in real time, and renders the resulting Markdown formatting as changes are made to the document. Built using MongoDB, Node JS, Express, socket.io, Handlebars, JQuery, Bootstrap, and Heroku.

HappyFinder

HappyFinder is a way to easily find happy hours by time-of-day and day-of-week. The project heavily utilizes data scraping / parsing of the Google Places and FourSquare APIs. Built with PostgreSQL, Python, Flask, Jinja, and Bootstrap. Deployed on an AWS instance with SSL.
