GitHub

SkillTreat is an eCommerce API built with PostgreSQL, Sequelize ORM, NodeJs, and Express. It's deployed on AWS and uses SSL encryption. The API features core eComm. capabilities (user, cart, order management, etc.). Authentication is stateless, using JSON web tokens. New user registration includes a verification flow, with users able to request new tokens upon expiration or blacklisting.
